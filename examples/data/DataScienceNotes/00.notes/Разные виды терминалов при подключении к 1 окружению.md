В дополнение к [[Заметка о разных оболочках|заметке о разных файлах логина]] стоит знать 1 факт - не все подключения ssh одинаково используют подобные файлы, яркий пример - jupyter hub и windows PWSH/VSCode + OpenSSH win:
 - при наличии `/etc/bash.bashrc` или аналога он выполнится при создании терминала в вэб jupyter-lab (например засетит переменные среды)
 - при подключении в контейнер через `docker exec -it <container> bash` аналогично произойдет полный логин в интерактивный терминал
 - при подключении по ssh к remote-server в дефольном случае не гарантировано, в качестве решения придется использовать специальный start-up скрипт, либо `source <file>` (последний вариант не гарантирует)
