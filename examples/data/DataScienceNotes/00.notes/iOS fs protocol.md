### WIN GUI option
___
https://www.3u.com/ - 3uTools - графический вариант доступа к FS айфона; не имеет CLI, соответственно подвязок к батч скриптам ОС "большой" машины.
### Ubuntu CLI interface
___
https://github.com/libimobiledevice/libimobiledevice - CLI опция для Ubuntu, работает также с подвязками к Cython для интеграции; возможно использовать как транспорт для скриптов автоматизации, нужны детали о безопасности.
